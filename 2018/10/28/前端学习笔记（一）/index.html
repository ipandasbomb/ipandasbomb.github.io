前端学习笔记（一）

前言：

一开始学习web建站，学习的是Python django开发后端，后来被前后端分离吸引，遂开始学习前端。

作为前端只熟悉html一些基本标签和css一些元素的菜鸟，自然去知乎查如何开始了，其中一个写的很好，看了一些视频。对前端三剑客也有了写基本的感受。

（一）概念了解

HTML：超文本标记语言

  • HTML：整个网页的骨架，定义了整个网页的基本框架结构。

CSS：层叠样式表

  • CSS：在HTML写好的基础上，对HTML元素进行个性化定制。

JavaScript：交互脚本语言

  • JavaScript：一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。

  • 个人理解JavaScript，是对网页中已经设计好的结构元素进行逻辑交互设计的语言。没有JavaScript，网页无法和人进行交互。

（二）知识详情

（一）CSS如何和HTML标签元素进行沟通？

通过选择器，CSS可以和HTML标签进行沟通联系。

  1. class类型选择器

  2. id选择器

    语法：用大于符号 > 表示选择器子选择器。用于设置子标签属性，注意是第一代后代属性
    .food > li {

    border: 1px solid red;

    }
    .food 表示类选择器下
    选择 li 标签，并进行设置
    ~~~

  3. 包含选择器

    与子选择器类型相似，但是将大于号 > 换位为空格，而且作用与所有后代。

  4. 通用选择器

    用 * 匹配HTML所有标签元素

  5. 伪类选择器

    很多种，典型常用的：
    a:hover {} 方法，表示鼠标划过时文本的显示属性。

  6. 分组选择符

    为多个标签元素设置同一个样式，可以用分组选择符，即逗号 “ , “ ，进行设计。比如：

    h1,span {

    font-size: red;
}

（四）实际设计

一个大概了解是有了，但是如何设计出一个完整的哪怕是简陋的网页呢？
